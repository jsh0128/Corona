# Corona(코로나 확장프로그램)

## 동기

확장프로그램을 개발하고 싶었는데 유용한 확장프로그램이 없을까 생각하다가 코로나 확장프로그램을 개발하게 되었습니다.

- 시/도 별 코로나 확진자 수, 사망자 수, 전일대비 증가수를 나타내 줍니다.
- 연령/나이 별로 확진자 수, 사망자 수, 확진율,사망률을 나타내줍니다.
- 대한민국에 있는 국민안심병원의 정보를 알려줍니다.
